# 僕の人工言語の文法

## 音韻論と表記法

- CV
- **何も**ひらがなで書かれる。
- 日本語の異音を避けるために「ち、つ、し」を省く。

|       | a   | e   | i   | o   | u   |
|-------|-----|-----|-----|-----|-----|
|       | `あ` | `え` | `い` | `お` | `う` |
| h     | `は` | `へ` | `ひ` | `ほ` | `ふ` |
| p     | `ぱ` | `ぺ` | `ぴ` | `ぽ` | `ぷ` |
| t     | `た` | `て` |      | `と` |     |
| k     | `か` | `け` | `き` | `こ` | `く` |
| m     | `ま` | `め` | `み` | `も` | `む` |
| n     | `な` | `ね` | `に` | `の` | `ぬ` |
| s     | `さ` | `せ` |      | `そ` | `す` |
